# Library Project

Простой проект на JavaScript для управления библиотекой книг.  

## Описание
Позволяет:
- Добавлять книги с названием, автором, количеством страниц и статусом прочтения.
- Отображать книги в таблице.
- Удалять книги.
- Менять статус прочтения.
- Данные сохраняются в `localStorage`, чтобы библиотека сохранялась при перезагрузке страницы.

## Как использовать
1. Открой `index.html` в браузере.
2. Заполни форму с данными книги и нажми кнопку **Add**.
3. Книга появится в таблице.
4. Для удаления книги нажми кнопку **Delete**.
5. Для изменения статуса прочтения нажми кнопку **Toggle Read**.
